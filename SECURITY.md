# Security Policy

## Supported Versions

| Version | Supported          | 
| ------- | ------------------ | 
| 1.x.x   | :white_check_mark: - Latest and updated version | 
| < 1.0   | :x: - Remove and install latest version [here](https://github.com/DavidTheRobloxDev/policy-service/releases) |
